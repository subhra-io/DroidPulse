# Optimizer Dashboard

Real-time web dashboard for monitoring Android app performance.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Live WebSocket connection to Android SDK
- Screen timing visualization
- API call monitoring
- Memory usage graphs
- FPS tracking
- Dark mode UI

## Requirements

- Node.js 18+
- Android app running with Optimizer SDK
- WebSocket server on port 8080
